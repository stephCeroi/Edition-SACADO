\begin{His}

 Pour représenter les nombres, nous utilisons de nos jours l'écriture
 décimale, mais ça n'a pas toujours été ainsi !

 Les égyptiens, par exemple, utilisaient des symboles pour un, dix,
 cent, mille, etc. :

\centerline{\Huge
{\pmhgfamily\Hone},~
{\pmhgfamily\Hten},~   
{\pmhgfamily\Hhundred},~
{\pmhgfamily\Hthousand},~
{\pmhgfamily\HXthousand},~
{\pmhgfamily\HCthousand},~
{\pmhgfamily\Hmillion}
}
 et pour représenter par exemple cent-vingt-trois :

\centerline{\Huge
{\pmhgfamily\Hone\Hone\Hone}{\pmhgfamily\Hten}{\pmhgfamily\Hten}{\pmhgfamily\Hhundred}
}

 Nous utilisons encore parfois les nombres romains : les symboles
 pour 1,5,10,50,100,500,1000 
 sont \Romanbar{1},\Romanbar{5},\Romanbar{10},L,C,D,M,
 et la valeur d'un symbole dépend un peu de sa position :
 le \Romanbar{1} dans \Romanbar{6} vaut 1, mais dans \Romanbar{4} il vaut $-1$.
 
\parbox[t]{0.6\textwidth}{ Les indiens dès le \Romanbar{5}\ieme siècle ont inventé un système très proche de
 notre système actuel, et les arabes, à partir du \Romanbar{7}\ieme siècle,
 ont développé des techniques de calcul et les ont diffusées en Europe,
 avec peu de succès. Les nombres non entiers sont encore réprésentés par des fractions.

Ce n'est qu'au \Romanbar{17}\ieme siècle, sous l'impulsion de Simon \textsc{Stévin},
 que les européens adoptent le système
 décimal pour représenter les nombres entiers mais aussi les nombres
 décimaux sans utiliser de fraction.}
\parbox[t]{0.39\textwidth}{

\end{center}

\section{L'ensemble des nombres décimaux $\mathbb{D}$}

\noindent\mini{0.48\textwidth}{
\begin{DefTQ}{Nombre décimal}{}
Un nombre décimal est un nombre qui peut s'écrire avec une partie décimale finie. 

L'ensemble des nombres décimaux est noté $\mathbb{D}$.
\end{DefTQ}
}{0.48\textwidth}{
\begin{ExTQ}{Nombres décimaux}{}
\[ 0~;~3~;~-5~;~7,33~;~\dfrac{3}{100}~;~\dfrac{21}{700} \]
sont des décimaux. Par contre, on peut démontrer que
$\dfrac 13$ ; $\sqrt 2$ ; $\pi$ ne sont pas des décimaux.
\end{ExTQ}
}

\noindent
\mini{0.70\textwidth}{
\begin{Pp}
Les nombres décimaux sont les nombres de la forme $\dfrac{a}{10^p}$ avec $a$ appartenant à $\Z$ et $p$ appartenant à $\N$.
\end{Pp}
}{0.28\textwidth}{
\begin{Rq}
$\mathbb{N}\subset\mathbb{Z}\subset\mathbb{D}$
\end{Rq}
}
\begin{Pp}
\textcolor{red}{Démonstration exigible}

$\dfrac{1}{3}$ n'est pas un nombre décimal.
\end{Pp}

\begin{Pv}
\underline{Par l'absurde}. On suppose que $\dfrac{1}{3}$ est un nombre décimal, alors par propriété :
$\ds \exists a\in\Z\,\text{ et }\,p\in\N\text{ tels que }\dfrac{1}{3}
=\dfrac{a}{10^p}$.

Donc $10^p=a\times3$ (produit en croix) donc $3$ est un diviseur de $10^p$ or $10^p=2^p\times5^p$ (décomposition en facteurs premiers). Donc $3$ ne peut pas diviser $10^p$ (l'hypothèse est donc absurde). Ainsi, on peut affirmer que $\dfrac{1}{3}$ n'est pas un nombre décimal.
\end{Pv}


\section{L'ensemble des nombres rationnels $\mathbb{Q}$}

\mini{.38\linewidth}{
\begin{Def}
Un nombre rationnel est un nombre qui peut s'écrire sous la forme $\frac{p}{q}$ avec $p$ appartenant à $\mathbb{Z}$ et $q$ appartenant à $\mathbb{N}^{\ast}$.

L'ensemble des nombres rationnels est noté $\mathbb{Q}$.
\end{Def}

\begin{Rq}
$\mathbb{N}\subset\mathbb{Z}\subset\mathbb{D}\subset\mathbb{Q}$
\end{Rq}

\begin{Pp}
\textcolor{red}{Démonstration exigible}

$\sqrt{2}$ n'est pas rationnel.
\end{Pp}
}{.58\linewidth}{
\begin{Pv}
\underline{Par l'absurde} :

On suppose que $\sqrt{2}$ est un nombre rationnel, alors par définition : 

\[\exists p\in\Z \text{ et }q\in\N^{\ast}\text{ tels que }\sqrt{2}=\dfrac{p}{q}\]

On suppose que $\dfrac{p}{q}$ est une fraction irréductible.

$\sqrt{2}=\dfrac{p}{q}$ donc $\sqrt{2}\times q=p$ donc $(\sqrt{2}\times q)^2=p^2$ donc $2\times q^2=p^2$ et $p^2$ est donc un nombre pair.

Si le carré d'un nombre est pair alors on peut affirmer que ce nombre est pair donc $p$ est un nombre pair : $\exists p_1\in\mathbb{N}\text{ tels que }p=2\times p_1$ donc $2\times q^2=p^2=(2\times p_1)^2=4\times p_1^2$. Donc $q^2=2p_1^2$ et $q^2$ est donc un nombre pair. Si le carré d'un nombre est pair alors on peut affirmer que ce nombre est pair donc $q$ est un nombre pair.

Si $p$ est un nombre pair et que $q$ est un nombre pair alors la fraction $\frac{p}{q}$ n'est pas irréductible, l'hypothèse est donc absurde.

Ainsi, on peut affirmer que $\sqrt{2}$ n'est pas un nombre rationnel.
\end{Pv}
}

\begin{Def}
L'ensemble des nombres réels est l'ensemble de tous les nombres que vous connaissez en seconde.

L'ensemble des nombres réels est noté $\R$.
\end{Def}

\begin{Rq}
$\mathbb{N}\subset\mathbb{Z}\subset\mathbb{D}\subset\mathbb{Q}\subset\mathbb{R}$
